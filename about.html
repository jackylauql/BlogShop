About

This personal project is an attempt to create a fully functional online blogshop that introduces a brand, provide news, updates and promotions of its' latest products as well as being a platform where buyers can purchase the products without going through a 3rd-party.

The project encompasses basic HTML, CSS and Javascript codes and is the first project I started for my target to create a portfolio of personal projects to further my career as a Full Stack Developer.
